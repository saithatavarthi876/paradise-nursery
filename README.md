# Paradise Nursery Shopping Application

Welcome to the Paradise Nursery Shopping Application! This project is a dynamic web application for an online plant shop, allowing users to browse a variety of houseplants, view details, add items to a shopping cart, and manage their purchases. The application features a modern UI, interactive cart management, and a seamless shopping experience.

## Features
- Browse plants by category
- View plant details (image, name, description, price)
- Add plants to a shopping cart
- Adjust item quantities and remove items
- Dynamic cart total calculation
- Responsive navigation bar

## Technologies Used
- React
- Redux Toolkit
- CSS
